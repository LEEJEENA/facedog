<h3><b>🎫 프로젝트 소개 🎫</b></h3>
 **반려견의 일상을 사진과 글로 기록하고, 다른 견주들과 소통할 수 있는 커뮤니티**

 - 상세 설명 : - 반려견의 일상을 찍은 사진과 반려견에 관한 글을 업로드 할 수 있습니다.
 - 글에서는 댓글을 통해 서로 소통할 수 있습니다.
